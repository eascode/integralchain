積分鏈。小積分.大夢想。


1.積分分散、不通用。不同行業，不同企業各自發行不同的積分，對積分的査詢和使用都不便捷，導致積分形同雞肋，沒有起到用戶導流的作用。

2.積分無法轉讓。根據統計，現時積分在國內的兌換率僅12%，近9成的用戶未被正確引導消費。

3.積分發行不透明。傳統的積分發行基於資料庫，資料維護成本高昂，而小企業的積分又存在篡改偽造的風險，使得公信力不足
